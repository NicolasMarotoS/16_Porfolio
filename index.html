Vino La Taca

Este proyecto consistía en realizar una marca para un producto de alta calidad. Se escogió el producto del vino ya que es un producto al que se le podía dar un gran cambio a las etiquetas del este producto ya que no es algo común. Se realizo una fase de investigación, una fase de bocetación y una fase final en la que colocar nuestro diseño en el producto, por lo cual se realizo una fase de fotografía mostrando el producto. Una pequeña muestra de cómo quedo el producto.
